# Flete Express Kangoo - Monterrey & San Pedro

Repositorio oficial del cotizador de fletes y mudanzas especializado en el área metropolitana de Monterrey.

## 🚀 Características
- **Cotizador Automático**: Cálculo de costos basado en la distancia entre el origen y el destino.
- **Branding Premium**: Interfaz moderna diseñada para clientes de San Pedro Garza García y Monterrey Sur.
- **Integración WhatsApp**: Generación automática de mensajes con todos los detalles del flete (Origen, Destino, Fecha, Hora).
- **Pagos**: Sistema de reserva con anticipo mediante Mercado Pago o Transferencia.

## 🛠️ Tecnologías
- HTML5 / CSS3 (Diseño responsivo)
- JavaScript (Lógica de cálculo radial)
- Google Fonts (Montserrat & Inter)

## 📂 Estructura
- `index.html`: Página principal del cotizador.
- `styles.css`: Estilos visuales premium.
- `fletes_kangoo_ad_creative.png`: Creativo publicitario para Facebook.
